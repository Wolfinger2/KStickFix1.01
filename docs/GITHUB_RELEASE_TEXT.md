# KStickFix 1.1

Saubere Basisversion ohne eigenen ISO-Schreiber.

## Funktionen

- USB-Sticks erkennen
- USB-Sticks analysieren
- USB-Sticks wiederherstellen
- externe ISO-Schreiber starten
- Auswahl zurücksetzen
- Warnung bei zu kleinem USB-Stick
- Diagnose
- Systemprüfung
