# KStickFix 1.0.1

KStickFix ist ein grafisches Linux-Werkzeug zum Analysieren, Wiederherstellen und Beschreiben von USB-Sticks.

## Enthalten

- USB-Stick-Erkennung
- Analyse
- Wiederherstellung
- exFAT, FAT32, NTFS, ext4
- MBR/GPT
- ISO-Schreiber
- Diagnose
- Systemprüfung
- KDE-Menü-Integration

## Projektstatus

Dieses Projekt gilt mit Version 1.0.1 als abgeschlossen. Der ursprüngliche Autor plant derzeit keine aktive Weiterentwicklung.
